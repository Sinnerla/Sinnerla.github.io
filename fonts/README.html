未来荧黑 (WeiLaiYingHei) Font

This theme uses WeiLaiYingHei Light weight.

How to get the font files

  1. Download the WeiLaiYingHei font from:
    https://github.com/celestialphineas/WeiLaiYingHei/releases

  2. Place these two files in this directory:

    • WeiLaiYingHei-Light.woff2
    • WeiLaiYingHei-Light.woff

  3. Or, modify source/css/style.css to use a different font.
