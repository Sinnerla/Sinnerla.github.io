VSCode 效率提升：让编码速度翻倍的技巧

2019-04-02 技术分享 #VSCode - 效率 - 编辑器

快捷键掌握

熟记快捷键是提升编码效率最直接的方式。

编辑类快捷键

  • Ctrl+D — 选中下一个相同单词
  • Ctrl+Shift+L — 选中所有相同单词
  • Alt+↑/↓ — 移动当前行
  • Shift+Alt+↑/↓ — 复制当前行

导航类快捷键

  • Ctrl+P — 快速打开文件
  • Ctrl+Shift+P — 命令面板
  • Ctrl+` — 打开终端

必备插件推荐

好的插件让 VSCode 如虎添翼。

GitLens

增强 Git 功能，显示行级别的 blame 信息和提交历史。

Prettier

自动格式化代码，保持团队风格一致。

json { "editor.formatOnSave": true, "editor.defaultFormatter": "esbenp.prettier-vscode" }

自定义 Snippets

重复的代码片段应该交给 Snippets 处理。

创建自定义 Snippet

在 File > Preferences > User Snippets 中配置。

好的 Snippet 应该包含常用的占位符和 Tab 停止点。
